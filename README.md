# AI-Powered Sales Forecasting & Business Intelligence System

##  Project Overview

This project analyzes retail sales data using Python, SQL, Machine Learning, and Power BI. It helps businesses understand sales performance, identify trends, and forecast future sales.

---

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- MySQL
- Power BI

---

##  Dataset

Sample Superstore Dataset

---

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Sales Forecasting using Machine Learning
- SQL Business Analysis
- Interactive Power BI Dashboard

---

##  Dashboard

Dashboard includes:
- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Sales by Region
- Profit by Category
- Monthly Sales Trend
- Top 10 Products
- Sales by Customer Segment

---

## Machine Learning

Model Used:
- Linear Regression

Objective:
- Forecast future monthly sales.
